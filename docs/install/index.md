The easiest way to install Coder is to use our
[install script](https://github.com/coder/coder/blob/main/install.sh) for Linux
and macOS.

<div class="tabs">

## Linux

```bash
curl -fsSL https://coder.com/install.sh | sh
```

## macOS

```bash
brew install coder/coder/coder
```
